# 典型要素

フェルマーの小定理

# 解法

`M^(K^N) mod P`を求めれば良い。MがPで割り切れる場合、答えは0。そうでない場合、フェルマーの小定理を利用する。`M^(P-1)=1`であるから、`M^(K^N mod P-1) mod P`が答え。
